# Dashboard de Benefícios (VT/VA) — Power BI

## Objetivo
Dashboard para monitorar **custos de benefícios** e **adesão (VT)**, com visão por **setor**, **funcionário**, **categoria** e evolução por **mês/ano**.

## Principais KPIs
- **% que aderem VT:** 15,80%
- **Funcionários que aderem VT:** 143
- **Total de funcionários:** 906
- **Total gasto:** R$ 87 mil

> Observação: valores e registros exibidos no portfólio devem estar **anonimizados**.

## Filtros e segmentações
- Setor
- Funcionário
- Categoria (Auxílio Creche, Bolsa Estudo, Gympass, Plano de Saúde, Vale Alimentação, Vale Transporte)
- Ano / Mês

## Regras de negócio
- Rateio de custo: **60% Funcionário** e **40% Empresa**.

## Entregáveis
- PDF do dashboard: `Beneficios.pdf`
- Imagens/prints: pasta `/images`

## Ferramentas
Power BI (modelagem + DAX) • Power Query • Excel/SQL (quando aplicável)
